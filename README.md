# NutriSnap 食刻 — Legal

Public hosting for the NutriSnap privacy policy, served via GitHub Pages.

- **Privacy Policy**: [index.html](index.html) → published at
  `https://<your-username>.github.io/nutrisnap-legal/`

This repository contains only the public legal page. The app source code lives
in a separate (private) repository.

To update the policy, edit `index.html` here (or re-copy it from the app repo's
`docs/privacy-policy.html`) and push — GitHub Pages redeploys automatically.
